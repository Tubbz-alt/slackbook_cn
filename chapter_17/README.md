包管理是任何Linux发行版的一个主要组成部分。Slackware包含的每一个软件以及很多第三方工具都是以源代码形式发布的，但编译这数以千计的程序和厍既麻烦又耗时，所以很多人都喜欢安装已经编译好了的软件包。事实上，在你安装Slackware的时候，安装程序主要做的就是在一系列的软件包上执行包管理工具。这里我们将会看到用于处理Slackware软件包的各种工具。
